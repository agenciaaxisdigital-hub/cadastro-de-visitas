# Cadastro de Visitas

Sistema de recepção e controle de visitas — Dra. Fernanda Sarelli.

## Stack

- React 18 + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase (auth + banco de dados)
- React Query + React Hook Form + Zod

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
